# Email Campaign Writer

Create emails that are easy to open, easy to skim, and built around one clear
next action.

## Workflow

1. Identify sequence goal and audience stage
2. Map the sequence arc across awareness, trust, and action
3. Write subject lines, preview text, and email bodies
4. Make each email distinct rather than repeating the same pitch

## Output Requirements

- subject line options
- preview text
- full body copy
- CTA
- sequence logic when writing multiple emails
