# Case Study Writer

Turn customer results into a proof asset that supports both trust and sales.

## Workflow

1. Capture the starting problem, implementation path, and result
2. Structure the story around challenge, approach, and measurable outcome
3. Use concrete metrics and believable detail
4. End with the strategic takeaway, not just the testimonial

## Output Requirements

- title
- customer context
- challenge
- solution
- results
- quote prompts or pull quotes if available
