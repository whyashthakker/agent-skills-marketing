# generative-engine-optimisation-geo

**A Claude Code skill for improving traditional SEO and GEO
(Generative Engine Optimisation) for websites and content.**

Use `/generative-engine-optimisation-geo` when you need keyword targeting,
schema guidance, AI-search visibility improvements, search audits, or
structured optimization plans.
