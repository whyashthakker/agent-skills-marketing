# SEO/GEO Tools and API Reference

Curated tools and utilities for search audits, structured data, and AI-search
visibility.

## Core Validation Tools

- Google Rich Results Test
- Schema.org Validator
- PageSpeed Insights
- Google Search Console
- Bing Webmaster Tools

## Useful Paid Platforms

- Ahrefs for backlink and keyword intelligence
- Semrush for broader SEO and AI-visibility workflows
- SE Ranking for affordable audit and rank tracking
- DataForSEO for API-driven SEO data pipelines

## GEO-Specific Monitoring Tools

- Profound
- Otterly.ai
- Peec AI
- Semrush AI visibility products

## Command-Line Checks

```bash
curl -sL "https://example.com" | grep -E "<title>|<meta"
curl -s "https://example.com/robots.txt"
curl -s "https://example.com/sitemap.xml"
curl -I "https://example.com"
```

## Robots.txt Pattern

```txt
User-agent: Googlebot
Allow: /

User-agent: Bingbot
Allow: /

User-agent: GPTBot
Allow: /

User-agent: ChatGPT-User
Allow: /

Allow: /

User-agent: ClaudeBot
Allow: /

```
