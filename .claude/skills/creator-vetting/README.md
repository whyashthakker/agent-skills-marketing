# creator-vetting

**A Claude Code skill for evaluating creators before outreach, briefing, or
payment.**

Use `/creator-vetting` when you need a defensible view of audience quality,
brand fit, content reliability, and risk.

## What it produces

- Vetting checklist
- Scored creator evaluations
- Red-flag analysis
- Approve / hold / reject recommendations
- Follow-up questions before contracting

## Usage

```bash
/creator-vetting Review these three TikTok creators for a wellness campaign
```

```text
"Help me vet a shortlist of creators before outreach"
"Build a brand safety and audience quality checklist for our influencer team"
```
