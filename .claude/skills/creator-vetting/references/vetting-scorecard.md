# Vetting Scorecard

## Core Dimensions

| Dimension | Score Range | What to Look For |
|----------|-------------|------------------|
| Audience fit | 1-5 | Right age, geography, interests, buying power |
| Engagement quality | 1-5 | Real comments, healthy save/share behavior |
| Content quality | 1-5 | Consistency, format fit, production standard |
| Brand safety | 1-5 | No obvious conflicts or reputational issues |
| Professionalism | 1-5 | Posting cadence, responsiveness, partnership maturity |

## Recommendation

- 22-25: approve
- 17-21: conditional approve
- 12-16: hold for more review
- below 12: reject
