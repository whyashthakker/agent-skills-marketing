# Event Promotion Planner

Coordinate event marketing so awareness, registration, and attendance all get covered. Use a phased timeline to avoid last-minute rushes.

## Quick Reference

**Key Insight:** Announce early for awareness, but don't try to close registration too soon. Vary the message as the event approaches rather than repeating the same announcement.

## When to Activate

Activate when the user asks to:
- plan promotion for an event
- drive event registration
- coordinate event marketing across channels
- create an event promotion timeline

## Workflow

### Step 1: Define Event and Audience

Gather or infer:
- event type (webinar, conference, workshop, launch)
- target audience
- channels (email, social, partners, paid)
- attendance or registration goal

### Step 2: Save The Date

Reference [references/promo-timeline.md](./references/promo-timeline.md):
- Announce early enough for awareness
- Don't try to close registration too soon
- Use teaser content: speakers, topics, format

### Step 3: Registration Push

Focus messaging on:
- audience problem the event solves
- speaker credibility and highlights
- event payoff (learn, network, exclusive access)

### Step 4: Reminder Cadence

Increase urgency as the event approaches:
- 2 weeks out: reminder + new angle
- 1 week out: last chance to register
- 1 day out: day-of logistics
- Day of: attendance prompt

Vary the message rather than repeating the same announcement.

### Step 5: Channel Mix

Map each phase to channels:
- **Email** – primary for registration, reminders
- **Social** – awareness, speaker highlights, countdown
- **Partners** – co-promotion, affiliate links
- **Landing page** – clear value prop, agenda, CTA

## Output Structure

- Event summary and goals
- Phased timeline (Save the Date → Registration Push → Reminders → Last Chance)
- Channel-by-channel plan
- Message angles per phase
- Landing page and email copy guidance

## Quality Gates

Before finalizing, verify:
- [ ] Timeline allows enough lead time
- [ ] Messages vary across phases
- [ ] Channels are assigned to phases
- [ ] Registration CTA is clear
- [ ] Day-of logistics are covered
