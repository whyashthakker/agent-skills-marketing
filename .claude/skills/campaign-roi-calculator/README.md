# campaign-roi-calculator

**A Claude Code skill for estimating or evaluating the financial performance of
creator and influencer campaigns.**

Use `/campaign-roi-calculator` when you need ROI, ROAS, CAC, EMV, or scenario
modeling for creator spend.

## What it produces

- ROI and ROAS models
- Scenario tables
- CAC and cost-per-output views
- Assumption summaries
- Decision-ready readouts

## Usage

```bash
/campaign-roi-calculator $25k creator campaign, 15 creators, estimate revenue and CAC
```

```text
"Model the ROI of a micro-influencer campaign for our skincare launch"
"Compare one macro creator vs ten micro creators financially"
```
