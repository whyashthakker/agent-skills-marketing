# Outreach Angles

## Co-Marketing

Useful when both brands serve overlapping audiences and can create content,
events, or distribution together.

## Integration

Best when product interoperability creates obvious user value.

## Referral Exchange

Works when the two businesses solve adjacent, non-competing needs.

## Content Collaboration

Good for webinars, reports, newsletter swaps, or interviews.

## Event Collaboration

Useful when each partner can contribute a different audience or asset.
