# Link Types

## Editorial Links

Best when you have data, original insight, or a genuinely useful resource.
These are the highest-quality links because the publisher chooses to cite you.

## Guest Contributions

Useful when the target site accepts expert articles and there is real audience
overlap. Only pursue if the content contributes actual value.

## Partner Links

Strong fit for integrations, agencies, service partners, and ecosystem pages.
These are often easier to secure and can be highly relevant.

## Resource Page Links

Useful for guides, templates, calculators, glossaries, or benchmark pages.
Match the asset to the curator's existing page structure.

## Data-Led Digital PR Links

Best for benchmarks, surveys, usage patterns, or trend reports that journalists
or newsletter writers can quote.
