# backlink-optimizer

Plans backlink acquisition around authority, relevance, and linkable assets
rather than generic link volume.

## What it helps with

- choosing which pages deserve links
- finding linkable asset ideas
- identifying target site types
- prioritizing outreach by difficulty and value

## Typical output

- backlink strategy summary
- priority pages
- asset recommendations
- outreach target categories
- short-term and mid-term actions
