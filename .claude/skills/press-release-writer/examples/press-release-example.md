# Press Release Example

Input:
"Write a press release announcing a new AI creator search feature for a
marketing platform."

Expected output shape:

- direct headline
- subhead with user value
- lead paragraph covering the announcement
- supporting section with feature detail and launch timing
- one executive quote
- short company boilerplate

Good answer qualities:

- reads like a release, not a landing page
- avoids hype-heavy adjectives
- gives media enough factual material to reuse
