# Ad Copy Generator

Write ad copy that matches platform behavior, audience awareness level, and the
offer's real differentiators.

## Workflow

1. Identify platform, audience temperature, and campaign objective
2. Choose 3 to 5 distinct creative angles
3. Write headline, body, CTA, and hook variants
4. Separate prospecting copy from retargeting copy

## Output Requirements

- angle-based ad concepts
- multiple headline and primary text variants
- platform-specific CTA suggestions
- short test plan if useful
