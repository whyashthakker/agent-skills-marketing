# pricing-page-copywriter

buyers choose confidently.

## What it helps with

- plan naming and framing
- comparison table copy
- objection-handling FAQ
- CTA wording
- pricing narrative for high-commitment pages

## Typical output

- pricing-page structure
- rewritten plan descriptions
- FAQ set
- CTA and proof suggestions
