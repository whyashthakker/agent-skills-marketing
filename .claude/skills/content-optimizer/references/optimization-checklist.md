# Optimization Checklist

## Core Checks

- Is the offer obvious in the first screen?
- Does the page clearly identify who it is for?
- Is there one primary CTA?
- Are proof points close to claims?
- Is the page easy to scan on mobile?

## Common Fixes

- replace abstract headline with concrete value
- move proof above the fold
- break dense text into sections
- add objection-handling FAQ
- tighten CTA copy so the next step is explicit

## Search/GEO Layer

- align headline and subheads with intent
- add short answer-first summary blocks
- include comparison tables where useful
- keep keywords natural
