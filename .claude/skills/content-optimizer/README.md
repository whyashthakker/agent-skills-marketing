# content-optimizer

**A Claude Code skill for improving existing copy on websites, landing pages,
product pages, and blog posts without replacing the underlying offer.**

Use `/content-optimizer` when the goal is better clarity, conversion, message
hierarchy, and search-readiness.

## What it produces

- Rewritten homepage and landing page copy
- Product page improvements
- Blog refreshes and structure cleanups
- CTA rewrites
- FAQ additions and trust-layer improvements

## Usage

```bash
/content-optimizer Rewrite our homepage hero and CTA for a creator analytics tool
```

```text
"Improve this landing page so the value proposition is obvious in the first screen"
"Optimize this article for readability and conversion"
```
