# blog-writer

**A Claude Code skill for writing strong blog posts, outlines, and content
refreshes for any website.**

Use `/blog-writer` when you need SEO-aware, GEO-aware blog content that is
clear, specific, and conversion-conscious.

## What it produces

- Blog outlines with title and angle options
- Full article drafts with H1/H2/H3 structure
- Product-led educational posts
- Comparison pages and how-to articles
- Refresh plans for outdated or underperforming posts

## Usage

```bash
/blog-writer B2B SaaS data pipeline platform, write a high-intent blog post on ETL vs ELT
```

```text
"Write a blog post for our agency on how to brief creators for a product launch"
"Turn this keyword into a detailed outline and draft"
```
