# Blog Post Templates

## 1. How-To Post

- H1: Clear desired outcome
- Intro: answer-first, who this is for, what they will learn
- H2: Why this matters now
- H2: Step-by-step process
- H2: Common mistakes
- H2: Checklist or summary
- H2: CTA

## 2. Comparison Post

- H1: Option A vs Option B
- Intro: quick recommendation by use case
- H2: Key differences
- H2: Feature or workflow comparison table
- H2: Best for
- H2: Pricing or effort tradeoffs
- H2: Final verdict

## 3. Product-Led Educational Post

- H1: Practical outcome or workflow
- Intro: teach first, pitch later
- H2: Problem framing
- H2: Current workflow
- H2: Faster implementation path
- H2: Mistakes to avoid
- H2: CTA tied to the workflow
