# creator-discovery

**A Claude Code skill for turning a brand goal into a creator sourcing brief
and shortlist strategy.**

Use `/creator-discovery` when you need to define creator criteria, tier mix,
platform fit, and discovery steps before outreach.

## What it produces

- Creator sourcing strategy
- Ideal creator profile
- Tier mix recommendations
- Search criteria and scorecards
- Discovery workflow for internal teams or agencies

## Usage

```bash
/creator-discovery DTC skincare launch, US women 22-34, TikTok and Instagram, prioritize UGC creators
```

```text
"Help me find creators for a new supplement launch"
"Build a sourcing brief for micro creators in fintech"
```
