# Question Types

## Screener

Use to filter respondents into the right audience or segment.

## Multiple Choice

Useful when you need clean comparison or segmentation-friendly answers.

## Rating

Best for satisfaction, importance, or confidence measures, but avoid stacking
too many rating questions in a row.

## Open Response

Use when language matters and you want pain-point wording in the customer's own
voice.

## Follow-Up Probe

Ask only when the prior answer creates a meaningful branch.
