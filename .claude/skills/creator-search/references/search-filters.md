# Search Filters

## Hard Filters

- platform
- geography
- language
- niche or content category
- follower range
- audience age or gender if required
- brand safety exclusions

## Preference Filters

- aesthetic fit
- posting cadence
- average engagement quality
- prior partnership categories
- creator responsiveness

## Search Logic

Start narrow on hard filters, then widen only one constraint at a time to avoid
polluting the longlist.
