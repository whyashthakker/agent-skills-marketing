# seo-content-brief

Builds search-intent-driven content briefs before drafting begins.

## What it helps with

- turning a keyword into a real article angle
- matching content to search intent
- structuring the outline
- identifying FAQs and internal-link ideas
- defining the right CTA for the reader stage

## Typical output

- primary keyword
- search intent
- recommended article type
- H2/H3 outline
- FAQ block
- CTA guidance
