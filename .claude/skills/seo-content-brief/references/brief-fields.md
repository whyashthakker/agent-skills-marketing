# Brief Fields

## Primary Keyword

The main query or phrase the page is trying to rank for.

## Search Intent

Clarify whether the user wants information, comparison, category exploration,
or a direct solution.

## Angle

Define what makes the content worth reading instead of repeating every other
result on the SERP.

## Outline

Recommend the H2/H3 structure based on the query type and reader stage.

## FAQs

Add high-intent and AI-search-friendly questions that extend the topic.

## CTA

Match the CTA to the article's role in the funnel.
