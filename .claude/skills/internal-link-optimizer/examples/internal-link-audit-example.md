# Internal Link Audit Example

Input:
"Review the sitemap for our marketing site and tell us which pages should
internally link to our influencer campaign analytics page."

Expected output shape:

- high-value target page list
- related source pages from the sitemap
- why each source page is relevant
- suggested anchor text directions
- priority order for implementation

Good answer qualities:

- recommendations are topic-driven, not arbitrary
- anchor text suggestions fit naturally into likely content
- the plan improves both navigation and SEO support
