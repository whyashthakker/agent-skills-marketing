# internal-link-optimizer

Builds sitemap-driven internal-link plans that connect relevant pages and
strengthen SEO without forcing unnatural links.

## What it helps with

- finding internal-link opportunities from sitemap URLs
- identifying orphan or weakly linked pages
- choosing which pages should link to money pages
- improving anchor text quality
- building topic authority through page relationships

## Typical output

- internal-link audit summary
- target page list
- source page recommendations
- anchor text suggestions
- prioritized implementation plan
