# Anchor Guidelines

## Good Anchor Text

- descriptive
- relevant to the target page
- readable in the sentence
- varied enough to avoid repetition

## Weak Anchor Text

- "click here"
- "learn more"
- exact-match repetition on every page
- anchors that overpromise what the target page contains

## Placement Principles

- body copy links are usually strongest
- add links where the target genuinely helps the reader
- keep the sentence natural
- avoid clustering too many links into one paragraph

## Linking Pattern Examples

- blog post about creator sourcing -> creator discovery feature page
- article about influencer ROI -> campaign analytics page
- comparison page -> pricing or demo page
